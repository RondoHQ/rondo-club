<decisions>
## Implementation Decisions

### Team Terminology
- Page title: "Teams" (same in Dutch)
- Industry field: Skip/hide for now (not needed)
- Parent Organization: Remove/hide (no longer relevant)
- Investors field: Translate to "Sponsoren" (sports club context)

### Commissie Terminology
- Page title: "Commissies"
- Role/position field label: "Rol"
- Chairperson role: "Voorzitter"
- Description field: "Omschrijving"
- Predefined roles: Minimal set — just "Voorzitter" and "Lid"

### Visibility & Status Labels
- Visibility options: "Openbaar" / "Privé"
- Active status: "Actief" / "Niet actief"
- Workspace label: Keep "Workspace" in English
- Show in directory toggle: "Toon in overzicht"

### Column Headers & Filters
- Members column: "Leden"
- Filter labels: "Alle teams" / "Actief" / "Gearchiveerd"
- Labels column: "Labels" (same in Dutch)
- Add buttons: "Nieuw team" / "Nieuwe commissie"

### Claude's Discretion
- Form field ordering and grouping
- Tooltip text and helper descriptions
- Error message phrasing
- Empty state messaging

</decisions>
