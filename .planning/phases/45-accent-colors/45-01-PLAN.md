<task type="auto">
  <name>Task 2: Update CSS component classes to use accent-* utilities</name>
  <files>src/index.css</files>
  <action>
Update the component classes in @layer components to use accent-* colors instead of primary-* for interactive elements:

1. .btn-primary: Change bg-primary-* to bg-accent-*, focus:ring-primary-* to focus:ring-accent-*
2. .btn-secondary: Keep as-is (uses primary-* for focus ring which is fine, but change focus:ring-primary-* to focus:ring-accent-*)
3. .input: Change focus:ring-primary-* to focus:ring-accent-*, focus:border-primary-* to focus:border-accent-*
4. .timeline-content a: Already uses accent-* for dark mode, update light mode to use accent-* too (text-primary-600 -> text-accent-600)

Note: Keep the Tailwind primary color definition in tailwind.config.js - it can stay as a fallback or for semantic uses. The accent-* CSS custom properties are what we're standardizing on for theming.
  </action>
